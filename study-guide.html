Study guide

Use every PDF like a study system.

A simple rhythm for turning long files into shorter, testable learning sessions.

1

Preview

Read the summary first, then scan headings and focus topics before deep study.

2

Question

Use suggested questions first, then ask the chat for objectives, procedures, key results, definitions, and weak points.

3

Recall

Close the PDF and explain the topic from memory. Use the answer page only to check gaps.

4

Repeat

Follow the dashboard reminder and revisit hard documents before exams or lab submission.
